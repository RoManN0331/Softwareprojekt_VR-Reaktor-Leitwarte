◆ calcDirectionCue()

void GazeGuidingPathPlayer.calcDirectionCue ( )

This method displays a red cue indicating to the player where to turn if the player is facing away from the currentTarget of the GazeGuidingPathPlayer in an angle > 30 degrees.

If the angle is > 160 degrees a second cue is being rendered indicating the object is behind the player. The Method calculates the alpha of the cues based on the angle between the player and the currentTarget of the GazeGuidingPathPlayer. If the current angle is smaller than the old angle the cue begins to fade out.
