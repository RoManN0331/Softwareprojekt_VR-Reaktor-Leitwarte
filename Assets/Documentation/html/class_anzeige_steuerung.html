Detailed Description

This class implements the basic functions to create displays that distinguish between three value ranges: normal (white), abnormal (orange), and critical (red).

This method animates the display's bar in three steps.

First the bar is set to indicate 100%. Then the bar is set to inndicate 0%. Finally the bar is set to indicate the actual value CHANGEpercentage.
